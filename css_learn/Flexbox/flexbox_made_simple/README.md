So what I learned so far:

Если поставить .nav-list li {
    width: max-content;
}
То это не позволяет контенту wrap'иться, то есть Не будет
```
Мурзин
Святослав
```
А будет просто 
```
Мурзин Святослав
```
Но! В случае, если мы ставим
.nav-list li {
    width: min-content;
}
То это будет wrappa'аться именно так:
```
Мурзин
Святослав
```
